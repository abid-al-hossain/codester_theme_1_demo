FAQ

Current answers for the source phase

Do I need a dev server?

Yes. This source project uses Vite module entrypoints, so development should run through `npm run dev`, and build verification should use `npm run build`.

Is the customizer shared across layouts?

Yes. Era, font, and palette preferences are stored once and reused as you move through the layout set.

Is this the shipping package?

No. This phase is only for source-product finalization. Release packaging and buyer-facing delivery structure are deferred to the later shipping phase.
