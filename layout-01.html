Core features

One system, many directions

Chronos brings layout variety, shared customization, and stable interactions into one flexible landing page system.

01

17 design eras

Chronos currently includes Modern, Gothic, Futuristic, Retro, Cyberpunk, Brutalist, Minimalist, Eco, Corporate, Artistic, Apocalyptic, Liquid, Jurassic, Pixelated, Ice Age, Volcanic, and Magic eras.

02

16 preset palettes

The shared customizer includes 16 preset palettes plus manual color controls, all persisted through local storage while you move between layouts.

03

70+ fonts

Heading, body, accent, and mono roles can be changed independently while you preview combinations in the shared customizer.

FAQ

Helpful answers before launch

How many eras are included?

Chronos includes 17 visual eras, each ready to reshape the same layout set with a different tone and typography profile.

Is the customizer shared across layouts?

Yes. Era, font, and palette preferences carry across the layout set so the system feels unified while you explore.

Can I export the current layout?

Yes. Package the active layout with its current era, colors, and fonts when you want a standalone version to keep building from.
